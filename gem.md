# role
tworzymy program do przeglądania plików z zadaniami pliki znajdują sie w folderze storage/app/private podzielone na galalog językowym np. php, javascript, html itp
program powinien umożliwić wygodne przeglądanie zadań i ich wyszukiwanie
każde zadanie zawiera plik zadanie.json oraz plik zadanie.md
plik zadanie.json zawiera informacje o zadaniu np. tytuł, opis, priorytet, data wykonania, status itp
plik zadanie.md zawiera opis zadania w formacie markdown
program powinien umożliwić przeglądanie bez edycji zadań z formatowaniem

serwis działa pod adresem https://mdview.t24.ovh/

strona główna pokazuje wszystkie kategorie zadań w formie kawelek
strona kategori pokazuje zadania z danej podkategorii. z wiltrami wylistowane
